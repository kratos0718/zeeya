# ZEEYA — Your AI Financial Brain

> Built by Teja & Abhinav · Visakhapatnam, India

Live at [zeeya.online](https://zeeya.online)

## Stack

- Next.js 14 (App Router)
- React Three Fiber — 3D scene
- Framer Motion — animations
- Tailwind CSS — styling

## Run locally

```bash
npm install
npm run dev
```

## Deploy

Hosted on Vercel. Connect `zeeya.online` domain in Vercel dashboard.
